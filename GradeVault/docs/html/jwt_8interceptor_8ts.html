Injectable ( )

HTTP interceptor for adding JWT authentication tokens to API requests.

This interceptor automatically adds the JWT authentication token to outgoing HTTP requests to the API. This ensures authenticated access without manually adding the token to each request.

Constructor for the JWT interceptor

Parameters
authServiceAuthentication service for accessing the current user's token

Intercepts HTTP requests to add authentication header when appropriate

Examines each outgoing request and adds the JWT token to the Authorization header if the user is logged in and the request is going to the API.
